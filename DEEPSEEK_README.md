# 🚀 DeepSeek 深度分析系统 - 快速入门

## 什么是 DeepSeek 深度分析？

这是一个集成了 **DeepSeek 大语言模型** 的乡村运营数据分析系统，能够自动对 122 个村庄的运营数据进行 **6 个维度的专业分析**，生成详细的分析报告。

## ⚡ 快速开始 (3 步)

### 1️⃣ 访问页面
```
http://localhost:8000/deepseek_analysis.html
```

### 2️⃣ 点击分析按钮
```
🔬 执行深度分析
```

### 3️⃣ 等待结果
```
30-60 秒后获得完整的深度分析报告
```

## 📊 分析报告包含

| 分析维度 | 包含内容 |
|---------|---------|
| **整体发展态势** | 总体状况、对标分析、发展趋势 |
| **区域差异分析** | 区域差异原因、经验总结、改进建议 |
| **业态结构优化** | 现状评估、推荐方案、差异化路径 |
| **收入提升路径** | 增收因素、成功经验、具体措施 |
| **风险管理防控** | 风险识别、影响评估、防控措施 |
| **创新发展建议** | 新兴业态、数字融合、品牌策略 |

## 🎯 三种导出方式

```
📥 导出报告    → TXT 文本文件下载
📋 复制全文    → 粘贴到 Word/邮件
🖨️ 打印报告    → 打印或保存为 PDF
```

## ❓ 常见问题

### Q: 分析需要多长时间?
**A:** 通常 30-60 秒，取决于网络和 DeepSeek 服务器。

### Q: 数据从哪里来?
**A:** 来自 `data/data.json` 文件，包含 122 个村庄的运营数据。

### Q: 分析结果准确吗?
**A:** 由 DeepSeek 大语言模型生成，具有专业的经济学基础。建议作为参考。

### Q: 可以离线使用吗?
**A:** 不能，需要网络连接才能调用 DeepSeek API。

### Q: 如何修改分析内容?
**A:** 编辑 `deepseek_analysis.js` 中的 `buildAnalysisPrompt()` 函数。
